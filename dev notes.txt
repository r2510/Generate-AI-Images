Quick and easy steps to start using JS, express and openai libraries,
1. install express openai libraries
2. generate your secret api keys from openai developer platform
3. initialize your key in .env file 
4. start server
5. create an instance of an Express Router and mount it at a certain path using router.post()
6. create an object for OPenAIApi class, and pass your secret key as object parameter for configuration.
7. create async function and call api using createImage function.
8. pass your text and size as parameter.
9. Render response on UI using HTML & CSS.
